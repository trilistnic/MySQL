База Данных course_project_SPDM описывает SPDM (Simulation Process & Data Management) систему хранения данных и их обработку.
В БД содержится следующая информация:
- о пользователях;
- данные для логина в систему;
- роль пользователя для настройки прав доступа;
- о созданных и харнимых проектах;
- состав команды по проекту из пользователей;
- настройка прав для проектной команды;
- 2 таблицы с основной информацией по моделям, загружаемым в систему;
- 2 таблицы с информацией по общим сборкам, которые формируются из моделей;
- 2 таблицы с информацией о расчетах и их текущем статусе, которые производятся на базе сборок;
- хранимые отчеты по результатам расчета;
- 2 таблицы с информацией по wiki страницам с полезнной информацией для пользователей;
- таблица для рассылки уведомлений на почту пользователем в случае, когда статус расчета изменилсяж
- таблица с логами о создании или ошибке при создании нового пользователя.

Для первых 3х таблиц создана процедура для автоматического заполнения всей необходимой информации о пользователе и внесении данных в лог файл.
Для моделей, сборок, расчетов, отчетов, хранимых в системе реализованы различные выборки и представления.
Для уведомления пользователя об изменении статуса расчета создан тригер, который заполняет таблицу для дальнейшей отправки писем.
Так же реализована процедура, которая выводит всю необходимую информацию о статусе расчета по логину пользователя.
